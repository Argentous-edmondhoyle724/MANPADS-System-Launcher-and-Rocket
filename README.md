## 📦 What Is This?

This project is a prototype of a low-cost rocket launcher and guided rocket system. It uses simple electronics and 3D-printed parts. The aim is to demonstrate how a small, guided rocket can be built and controlled.

You don't need to know anything about programming or electronics to try this software.
